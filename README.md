# Superhuman AUR

Unofficial AUR autopublisher for of [Superhuman](https://superhuman.com) email client.

## Installation

AUR helpers:

```bash
paru -S superhuman

yay -S superhuman
```

Manual:

```bash
git clone https://aur.archlinux.org/superhuman.git
cd superhuman
makepkg -si
```
